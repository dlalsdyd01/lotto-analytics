# 📢 로또 분석 웹앱 광고 수익화 가이드

## 🎯 추가된 광고 위치

현재 템플릿에 **4개의 광고 영역**을 추가했습니다:

1. **상단 배너** - 모든 사용자가 첫 화면에서 봄 (최고 노출)
2. **통계 탭 상단** - 통계 분석을 보는 사용자 타겟
3. **예측 탭 상단** - 가장 중요! 예측 번호를 보려는 핵심 사용자층
4. **하단 배너** - 스크롤 하단 도달 시 노출

---

## 1️⃣ Google AdSense 신청 및 설정

### Step 1: AdSense 계정 생성
1. [Google AdSense](https://www.google.com/adsense) 접속
2. Gmail 계정으로 로그인
3. "시작하기" 클릭
4. 웹사이트 URL 입력: `https://lottoanalytics.co.kr`

### Step 2: 사이트 등록 및 코드 삽입
AdSense에서 제공하는 **Head 코드**를 받으면:

```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
     crossorigin="anonymous"></script>
```

이 코드를 `templates/index.html`의 `<head>` 섹션에 있는 주석을 풀고 삽입하세요.

### Step 3: 승인 대기
- 승인 기간: **1~2주** (사이트 품질에 따라 다름)
- 승인 조건:
  - ✅ 오리지널 콘텐츠 (로또 분석 = OK)
  - ✅ 트래픽 존재 (최소 방문자 필요)
  - ✅ 정책 준수 (도박 X, 로또 분석/통계 O)

### Step 4: 광고 단위 생성
승인 후 AdSense 대시보드에서:
1. "광고" → "광고 단위" → "디스플레이 광고" 선택
2. **반응형** 선택 (모바일/PC 자동 대응)
3. 광고 코드 복사

예시:
```html
<ins class="adsbygoogle"
     style="display:block"
     data-ad-client="ca-pub-1234567890123456"
     data-ad-slot="9876543210"
     data-ad-format="auto"
     data-full-width-responsive="true"></ins>
<script>
     (adsbygoogle = window.adsbygoogle || []).push({});
</script>
```

이 코드를 각 광고 영역의 주석 부분에 삽입하세요.

## 2️⃣ 광고 코드 적용 방법

### 현재 템플릿에서 할 일:

#### 1. Head 코드 활성화
`templates/index.html`의 **9-11번째 줄** 주석 해제:
```html
<head>
    ...
    <!-- Google AdSense (승인 후 여기에 코드 삽입) -->
    <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-YOUR_ID"
         crossorigin="anonymous"></script>
</head>
```

#### 2. 각 광고 영역 활성화
템플릿에서 `<!-- AdSense ...` 주석을 찾아 해제하고 본인의 광고 코드 삽입:

**광고 영역 1 (상단 배너)**
```html
<div class="ad-container" style="text-align:center; padding:20px 0; margin-bottom:20px;">
    <ins class="adsbygoogle"
         style="display:block"
         data-ad-client="ca-pub-YOUR_ID"
         data-ad-slot="SLOT_ID_1"
         data-ad-format="auto"
         data-full-width-responsive="true"></ins>
    <script>
         (adsbygoogle = window.adsbygoogle || []).push({});
    </script>
</div>
```

**광고 영역 3 (예측 탭 - 가장 중요!)**: 동일한 방식으로 적용

- 한국 사용자 대상으로 수익성 좋음

**통합 방법:**
```html
<!-- 카카오 애드핏 -->
<ins class="kakao_ad_area"
     style="display:none;"
     data-ad-unit="YOUR_UNIT_ID"
     data-ad-width="320"
     data-ad-height="100"></ins>
<script type="text/javascript" src="//t1.daumcdn.net/kas/static/ba.min.js" async></script>
```

- Google AdSense 대안 (Yahoo/Bing 광고)
- URL: https://www.media.net

### 🔹 Propeller Ads
- 승인 거의 100%
- 팝업/배너 광고 (사용자 경험은 떨어질 수 있음)

## 4️⃣ 광고 배치 Best Practice

### ✅ 권장 위치
1. **예측 탭 상단** - 사용자가 예측 번호를 보기 직전 (전환율 최고!)
2. **상단 배너** - 첫 화면 노출
3. **콘텐츠 중간** - 통계 차트 사이사이

### ❌ 피해야 할 것
- 너무 많은 광고 (사용자 이탈)
- 당첨확인 버튼 근처 (오인 클릭 유도는 정책 위반)
- 로딩 중 화면에 광고 (UX 저하)

### 📊 예상 수익
- 일 방문자 100명: 월 $5~20
- 일 방문자 1,000명: 월 $50~200
- 일 방문자 10,000명: 월 $500~2,000

(클릭률 및 CPC에 따라 변동)
